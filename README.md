# Punjab Judicial Workload Analysis

## Project Overview

This project analyzes the judicial workload and court efficiency across Punjab district courts using large-scale case-level judicial data from 2010–2018.

The objective is to identify workload distribution, pendency trends, case resolution patterns, and district-level judicial performance using data analytics and visualization.

---

## Problem Statement

Indian courts handle massive volumes of pending and resolved cases. Understanding workload distribution and delays can help identify bottlenecks in judicial efficiency.

This project focuses specifically on Punjab district courts to uncover actionable insights regarding:

* District-wise case workload
* Pending vs resolved cases
* Case disposal efficiency
* Resolution time trends

## Dataset Source

The raw judicial dataset used in this project was obtained from the Development Data Lab (DDL) Judicial Data Portal.

**Original Dataset Source:**
https://www.devdatalab.org/judicial-data

Dataset Highlights:

* Covers **81 million district court cases across India**
* Time Period: **2010–2018**
* Includes filing dates, decision dates, court details, district information, and case status
